% This command prints the sub-title and is completely optional.
\cscsubtitle{Finally a LAN!}

We have been given permission to have a LAN using the computer labs.
Games such as \emph{Wolfenstein: Enemy Territory} and other free games will be played.\\
All games will be provided.\\
\\
This is a \textbf{FREE} event and anyone is welcome. The CSC will be open for snacks and drinks as usual.
